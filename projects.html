My Projects

Here are a few of the many projects I have worked on.

NOOK

NOOK Decor Inspiration Platform is a user-friendly platform that allows users to explore inspiring interiors through unique moodboards. The demo version of the application showcases core functionalities like moodboard previews and image interactions, although some features may not be fully implemented. The project was developed as part of the Co.Lab program, emphasizing the importance of collaboration with designers, product managers, and other developers.

Chat Application Project

A real-time chat application built using Flask-SocketIO for the backend and React with TypeScript for the frontend. The app enables users to exchange messages in real-time through WebSocket technology. Developed as part of a homework assignment during the Coding Temple bootcamp, this project demonstrates my proficiency in full-stack web development, including both front-end and back-end technologies.

Personal Dashboard Application with GraphQL

A dynamic user dashboard built with React, TypeScript, and GraphQL. This application allows users to manage their profile data, including posts, albums, and todos. It features CRUD operations, a responsive design, and an intuitive user interface. Developed as part of a bootcamp assignment, the project demonstrates expertise in GraphQL queries and mutations, enhancing user interaction and experience.

Task Management Application with Typescript Project

This is a task management web application built using TypeScript and React. Users can create, edit, delete tasks, and view detailed information for each task. The application implements authentication and authorization with Auth0. It supports form validation and global state management through the Context API. The project demonstrates the application of key TypeScript principles such as strict typing, state management using Typed Hooks, and the use of interfaces to improve data security and code readability.

PokeAPI Integration Project
