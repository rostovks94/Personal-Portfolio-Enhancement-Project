PokeAPI Integration Project

PokeAdventure is an interactive web application designed for Pokémon enthusiasts. This project allows users to explore detailed information about their favorite Pokémon, including abilities, types, stats, and evolutionary paths. It features a search functionality where users can input a Pokémon name to fetch and display comprehensive data from the PokeAPI. Additionally, users can simulate battles to see how different Pokémon perform in combat scenarios.

Task Management Application

The Task Management Application is a user-friendly and efficient tool designed to help users manage their tasks effectively. Developed using modern web technologies, this application features a responsive design, intuitive interface, and seamless user experience. It allows users to create, update, and delete tasks, set priorities, and track progress. The application is built with a focus on usability and performance, ensuring that users can easily organize their workload and stay productive.
